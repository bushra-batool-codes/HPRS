# Hospital Patient Record System (HPRS)

## Overview

A Python-based system to manage hospital patients, admissions, billing, and analytics efficiently using structured programming and data analysis tools.

## Features

* Add and manage patient records
* Admit patients to wards with capacity tracking
* Discharge patients with automatic billing
* Store data using CSV files
* Perform analytics on hospital data
* Generate visual reports (Bar, Pie, Line charts)

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib

## How to Run

1. Install dependencies:
   pip install pandas numpy matplotlib

2. Run the program:
   python cli.py

## Outputs

* Ward Occupancy (Bar Chart)
* Patient Distribution (Pie Chart)
* Peak Admissions (Line Graph)

## Project Structure

* models.py → Class definitions (OOP)
* utils.py → Validation logic
* storage.py → File handling (CSV)
* analytics.py → Data analysis
* visualization.py → Graphs
* cli.py → Main interface

## Key Concepts Used

* Object-Oriented Programming (OOP)
* File Handling
* Data Analysis (Pandas, NumPy)
* Data Visualization (Matplotlib)
